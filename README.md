# Project-1-The "Mythbuster" Meta-Analysis.
Abstract:
Time-Restricted Eating (TRE) has surged in popularity as a non-pharmacological intervention for metabolic syndrome. However, individual randomized controlled trials (RCTs) show conflicting results regarding its efficacy in lowering HbA1c compared to standard caloric restriction. This project pools data from simulated RCTs to perform a meta-analysis, visualizing the synthesized effect size via a forest plot to determine if TRE offers a statistically significant clinical advantage.
